Detailed Description

Abstract representation of the world.

The world consists of a number of fields, that are ordered in a coordinate system. A routine assigns the fields to different threads, that move the fields from current state to the next state. Moving a field from one state to another means increasing age of creatures, growing plants, move creatures from one field to another, ...

void ppsim::AbstractWorld::phase_model ( long  thread_id)

Thread function (implements phase model).

Implements phase modell:

  1. Phase 0
    • update plant config objects
    • update prey config objects
    • update predator config objects
  2. Phase 1
    • breed plants
    • eat plants
    • eat preys
  3. Phase 2
    • choose direction
  4. Phase 3
    • move creatures
  5. Phase 4
    • finish moving creatures
  6. Phase 5
    • increase age of organisms
    • make creatures hungry
    • generate offsprings

idand pointer to the object
